## ABS Function

The `ABS` function in Excel returns the absolute value of a number. The absolute value of a number is its value without its sign, making any negative number positive.

### Syntax:

    ABS(number)

- **number**: The number for which you want the absolute value.

### Examples:

1. `=ABS(-5)` would return 5.
2. `=ABS(8)` would return 8.
3. `=ABS(A1)` would return the absolute value of the number in cell A1.

> **Note**: The `ABS` function is particularly useful when you need to calculate distances, differences, or other metrics where only the magnitude matters and not the direction (positive or negative).
