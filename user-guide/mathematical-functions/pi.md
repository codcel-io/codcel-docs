## PI Function

The `PI` function in Excel returns the mathematical constant π (pi), which is approximately 3.14159. This function is
useful in mathematical, engineering, and scientific calculations where you need to work with the value of π.

### Syntax:

    PI()

This function does not take any arguments and simply returns the value of π.

### Examples:

1. `=PI()` would return the value `3.14159265358979`.

2. If you want to calculate the circumference of a circle with a radius of 5, you can use the formula for circumference,
   `2 * π * radius`, in Excel: `=2 * PI() * 5` would return `31.4159265358979`.

> **Note**: The `PI` function is fundamental in formulas involving circular or spherical calculations, useful in fields
> such as geometry, trigonometry, and physics.
